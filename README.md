# Char.io

This is Third Person Shooter game.

## Tooling

*Webpack 4.x
*BabylonJS 3.x
*Babel 7.x

## How to launch it:

Check out this repository.

`git clone https://github.com/ZyromM/Char.io.git`

`yarn install or npm install`

`yarn start or npm start`

It will be listening on `localhost:9000`
